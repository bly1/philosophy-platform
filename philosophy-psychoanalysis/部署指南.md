## 🌐 部署方案

### 方案1：Netlify Drop（最简单）
1. 打开 [https://app.netlify.com/drop](https://app.netlify.com/drop)
2. 拖拽 `哲学学习平台-完整版.html` 文件到页面
3. 立即获得免费网址：`https://random-name-123456.netlify.app`

### 方案2：GitHub Pages
1. 创建新 GitHub 仓库
2. 上传 HTML 文件
3. 在 Settings → Pages 中启用：
   - Source: Deploy from a branch
   - Branch: main
   - 文件夹：/(root)
4. 访问：`https://username.github.io/repo-name`

### 方案3：Vercel
1. 登录 [https://vercel.com](https://vercel.com)
2. 点击 New Project
3. 拖拽 HTML 文件
4. 自动部署完成

### 方案4：Firebase Hosting
1. 安装 Firebase CLI：`npm install -g firebase-tools`
2. 运行：`firebase init hosting`
3. 上传 HTML 文件到 public 文件夹
4. 部署：`firebase deploy`
