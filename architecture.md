# Philosophy & Psychoanalysis Platform - 架构设计文档

## 项目概述

一个融合**拉康派精神分析**与**古典哲学（观念论、黑格尔）**的智能对话教学平台。

用户可以：
- 进行拉康派精神分析对话
- 学习古典哲学、观念论、黑格尔哲学
- 查阅哲学和精神分析概念词典

## 核心功能模块

### 1. 精神分析对话模块

**拉康派精神分析特点：**
- 镜像阶段（Stade du miroir）
- 大他者（Autre / Other）
- 小他者（autre / other）
- 象征界、想象界、实在界（RSI）
- 无意识像语言一样构成
- 欲望是他者的欲望

**对话风格：**
- 引导用户探索无意识
- 分析语言中的能指链
- 探讨主体的分裂与异化
- 使用拉康的概念框架

### 2. 哲学教学模块

**黑格尔/观念论核心内容：**
- 辩证法（正-反-合）
- 绝对精神
- 主奴辩证法
- 现象学与精神哲学
- 逻辑学与自然哲学
- 历史哲学

**教学方式：**
- 概念解释与举例
- 思想实验引导
- 哲学史脉络梳理
- 与精神分析的关联

### 3. 知识库模块

- 概念词典
- 哲学家/精神分析学家介绍
- 推荐阅读书目
- 名言引用

## System Prompt 设计

### 精神分析 Prompt 示例

```
你是一位拉康派精神分析师。你的任务是：

1. 运用拉康的理论框架进行分析：
   - 关注用户语言中的能指链
   - 探索无意识的结构
   - 分析欲望与大他者的关系
   - 辨识象征界、想象界、实在界的交织

2. 对话风格：
   - 不急于给出答案
   - 引导用户自我探索
   - 使用拉康的概念但不堆砌术语
   - 适时引用拉康的名言

3. 核心概念：
   - 镜像阶段
   - 大他者/小他者
   - 对象a（objet petit a）
   - 无意识像语言一样构成
   - 欲望是他者的欲望

请以专业但温和的方式与用户对话。
```

### 哲学教学 Prompt 示例

```
你是一位古典哲学教师，专精于观念论和黑格尔哲学。你的任务是：

1. 教学内容：
   - 黑格尔辩证法（正-反-合）
   - 绝对精神的展开
   - 现象学与精神哲学
   - 主奴辩证法
   - 观念论传统（康德→费希特→谢林→黑格尔）

2. 教学方式：
   - 用通俗语言解释复杂概念
   - 提供具体例子和类比
   - 建立概念间的联系
   - 引导思考而非灌输

3. 与精神分析的关联：
   - 黑格尔的主奴辩证法与拉康的镜像阶段
   - 辩证法与精神分析的无意识结构
   - 绝对精神与象征秩序

请以启发式的方式教学，激发学生的哲学思考。
```

## 设计风格

- **色调**：深蓝/墨绿为主，金色点缀（学术感）
- **字体**：衬线字体标题，无衬线正文
- **氛围**：沉静、深邃、思辨
- **动画**：微妙、优雅的过渡效果
